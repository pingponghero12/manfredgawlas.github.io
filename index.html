My Projects

CFD simulaions for Attitude Control System

Curently researching optimal domain and mesh for 4 parameter study of Sternik 1.5 rocket.

I'm currently working in ANSYS to conduct parametric study of forces and torques acting on rocket depending on angles of attack and side slip, as well as independent angles of control surfaces. The goal is to find aerodynamic coefficients for rocket which we could insert into our control algorithm.

For this project I'm using ANSYS Fluent to conduct simulations. I'm also using Solidworks and Ansys SpaceClaim to prepare geometries for simulations.

We already made several partial studies using Solidworks Fluid Simulations and Ansys Fluent, now we are waiting for full Ansys licence to preform full study on computing cluster.

Drag Coefficient Study

I conducted a study of drag coefficient as a function of Mach number for a rocket model that I created based on PWR is Space's R5 rocket. Results were compared with experimental data from Modern Exterior Ballistics and with results from other simulations. The goal was to find out how accurate are the results of simulations were and to find graph for this rocket's drag coefficient.

Documentation of this project on github

Water Rocket Simulator

I'm working on rewriting it, so it meets standards of OOP.

This application simulates simple rocket engine, consisting of chamber with pressurized gas and liquid. Rocket thrust is generated through expulsion of either liquid or gas, caused by the expansion of gas. Users have the flexibility to select from a list of predefined gases or introduce custom ones.

Beyond individual simulations, the program offers a feature allowing users to visualize plots depicting output values against varying input parameters.

To learn more about project I reccomend you refer to repository and to documentation

GitHub repository Documentation of simulator

RMS2

RSM2 is a simulator of a stable fuel rocket engines for BATES geometries. The program allows users to get graphs of pressure, thrust as well as values of impulse and time of burning.

Program is writen in python with usage of matplotlib.
